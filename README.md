> This is a authentication example with Golang, PostgreSQL, Redis.

## Usage


```sh
# clone repository
git clone https://github.com/Myoschen/golang-auth.git

# change directory
cd golang-auth

# run docker
docker compose up -d

# run api
go run cmd/main.go
```
